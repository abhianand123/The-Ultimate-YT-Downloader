# The Ultimate YT/YTMusic Downloader

A beautiful web-based YouTube and YouTube Music downloader.

## Requirements

- **Python 3.8+**
- **FFmpeg** (required for audio conversion)
  - Windows: Download from [ffmpeg.org](https://ffmpeg.org/download.html) and add to PATH
  - Mac: `brew install ffmpeg`
  - Linux: `sudo apt install ffmpeg`

## Installation

```bash
pip install -r requirements.txt
```

## How to Run

```bash
python app.py
```

> If `python app.py` gives an error, try `py app.py`

Then open **http://localhost:5000** in your browser.

## Features

| Feature | Description |
|---------|-------------|
| YouTube Video | Download in any quality (1080p, 720p, etc.) |
| YouTube Music | Extract audio as MP3 (320kbps) |
| Playlist Download | Download entire playlist as ZIP |
| Quality Selection | Choose from all available qualities |
| Real-time Progress | Live download progress with speed |
| Auto Cookie Support | Bypasses 403 errors using browser cookies |
